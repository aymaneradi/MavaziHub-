\# MavaziHub



MavaziHub ist eine webbasierte B2C-E-Commerce-Plattform für traditionelle und modern interpretierte afrikanische Kleidung.



\## Projektstruktur



\- `backend/` – Java Spring Boot Backend

\- `frontend/` – React + CoreUI Frontend

\- `docs/` – technische Dokumentation, ADRs und Diagramme

\- `database/` – Datenbankschema, Seed-Daten und spätere Migrationen



\## Technologie-Stack



\- Backend: Java Spring Boot

\- Frontend: React + CoreUI

\- Datenbank: PostgreSQL

\- Containerisierung: Docker / Docker Compose

\- Versionsverwaltung: GitHub



\## Lokaler Start



Die konkrete Startanleitung wird im Verlauf der Implementierung ergänzt.



\## Teamarbeit



Die Entwicklung erfolgt über Git-Branches und Pull Requests. Der Branch `main` enthält stabile Versionen. Die laufende Entwicklung erfolgt über `develop` und Feature-Branches.
